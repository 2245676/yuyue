# UI美化改造方案

基于参考图片（食べログノート风格）的详细分析和实施计划。

## 参考图片分析

### 共同特征
1. **顶部日期导航**
   - 圆角背景（灰色/白色）
   - 左右箭头按钮
   - 日期显示（1月4日 (日)）
   - 右侧：+ 按钮和刷新按钮

2. **标签页切换**
   - タイムスケジュール（时间表）- 橙色下划线表示选中
   - リスト（列表）- 灰色未选中
   - 简洁的文字标签，无边框

3. **预约卡片**
   - 橙色渐变背景（#FFA726 到 #FFB74D）
   - 圆角设计（border-radius: 12px）
   - 无边框或细边框
   - 柔和阴影效果
   - 显示客户名称和人数

4. **非营业时间**
   - 灰色背景（#F5F5F5）
   - 斜杠纹理（diagonal stripes）

5. **底部浮动按钮**
   - 蓝绿色背景（#26A69A / teal）
   - 大圆角（border-radius: 24px）
   - 白色文字
   - 图标+文字
   - 柔和阴影
   - 文字：「本日のネット予約を止める」

6. **底部导航栏**
   - 4个图标按钮：
     * 予約一覧（预约列表）- 橙色选中
     * ブロック（屏蔽）
     * 通知一覧（通知列表）- 红点提示
     * その他（其他）- 红点提示
   - 图标+文字标签
   - 选中项有下划线

## 颜色方案

### 主色调
- **橙色**：#FFA726 (主色), #FFB74D (渐变), #FF9800 (强调)
- **蓝绿色**：#26A69A (按钮), #00897B (hover)
- **灰色**：#F5F5F5 (非营业时间), #E0E0E0 (边框), #9E9E9E (文字)

### 阴影
- 卡片阴影：`box-shadow: 0 2px 8px rgba(0,0,0,0.1)`
- 浮动按钮阴影：`box-shadow: 0 4px 12px rgba(38,166,154,0.3)`

## 圆角规范
- 预约卡片：12px
- 日期导航背景：24px
- 底部浮动按钮：24px
- 标签页下划线：2px（高度）
